# 💕 Moruvi

**Moruvi** 是一款專為情侶設計的互動式 APP，透過共享任務、日常互動與紀錄功能，讓彼此在忙碌生活中仍能維持連結與陪伴。

無論是完成共同目標、發布小任務、記錄點滴，還是增加情侶之間的趣味互動，Moruvi 希望成為陪伴每段關係成長的數位空間。

---

## ✨ Features

### 📝 里程碑紀錄
- 保存彼此完成過的任務
- 回顧共同成長歷程
- 建立專屬於兩人的回憶

### 📌 任務系統（Mission System）
- 發布專屬任務給另一半
- 任務批准 / 駁回機制
- 任務完成與歷史紀錄
- 任務狀態同步更新

### ☁️ 雲端相簿（Cloud Album）
- 上傳與保存兩人的照片與回憶
- 建立專屬於情侶的共享相簿
- 隨時回顧共同經歷的重要時刻
- 雲端同步，跨裝置查看珍貴紀錄

### 🔐 帳號與配對系統
- 使用者登入與驗證
- 情侶配對功能
- 雙方資料同步管理

### 📲 即時通知（Push Notifications）
- 訂閱個人化通知與事件提醒
- 接收任務批准、駁回與完成通知
- 即時同步情侶互動與系統更新
- 支援瀏覽器與行動裝置推播提醒

## 🛠 Tech Stack

Frontend:
- HTML / CSS / JavaScript
- Vue

Backend:
- Node.js
- Express.js
- MongoDB + Mongoose

Authentication:
- Token Authentication

Deployment:
- Docker
- Linux Server

## 🚀 Getting Started

Clone project:

```bash
git clone https://github.com/your-repository/Moruvi.git
cd Moruvi
```

frontend side:

```bash
npm install
npm run serve
```

backend side:

```bash
cd backend
npm install
node index.js
```
